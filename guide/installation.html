Instalação

Compatibilidade

O Stellar faz uso de todo o potencial do ECMAScript 6, por isso, versões do Node.js inferiores à 6 não são suportadas.

Release Notes

Os detalhes das releases para cada versão estão disponíveis no GitHub na aba Releases e no ficheiro de Changelog.

NPM

O NPM é o método recomendado para a instalação do Stellar, uma vez que este é usado para satisfazer as dependências não só do core como dos módulos.

Versões de Desenvolvimento

Para usar a versão de desenvolvimento do Stellar apenas tem que fazer o clone do repositório do GitHub. O branch master contem a ultima versão estável da framework, já a versão de desenvolvimento encontra-se no branch dev.

Nota: O npm link pode necessitar de permissões de administrador.
